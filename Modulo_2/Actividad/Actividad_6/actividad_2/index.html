Descripción del Curso

Bienvenido al curso de Desarrollo de Aplicaciones Full Stack con JavaScript! Este curso está diseñado para ofrecerte una formación integral en la creación de aplicaciones web modernas, utilizando JavaScript en el front-end y el back-end.

A lo largo del curso, aprenderás a desarrollar aplicaciones completas desde cero, cubriendo todos los aspectos necesarios para construir aplicaciones robustas y escalables.

Objetivos del Curso

El contenido de este libro se estructura en tres apartados bien diferenciados:

  • Comprender los fundamentos de JavaScript: Aprende los conceptos esenciales de JavaScript, incluyendo ES6+ y cómo aplicarlos en el desarrollo web.
  • Desarrollar aplicaciones web del lado del cliente: Utiliza frameworks y bibliotecas como React.js o Vue.js para crear interfaces de usuario interactivas y atractivas.
  • Construir el back-end con Node.js y Express.js: Aprende a configurar servidores, manejar peticiones y construir APIs RESTful utilizando Node.js y el framework Express.js.
  • Gestionar bases de datos: Aprende a interactuar con bases de datos tanto SQL (como PostgreSQL) como NoSQL (como MongoDB) y a integrar estas bases de datos en tus aplicaciones.
  • Implementar autenticación y seguridad: Descubre cómo implementar sistemas de autenticación y autorización, así como prácticas de seguridad para proteger tus aplicaciones.
  • Realizar despliegue y mantenimiento: Aprende a desplegar tus aplicaciones en entornos de producción y a mantenerlas, utilizando herramientas como Docker y plataformas de despliegue en la nube.
